## üé¨ Video Strategy

### Recommended Videos to Create:

1. **Homepage Hero Video** (Priority: HIGH)
   - Duration: 30-60 seconds
   - Content: Overview of programs, students in training, success stories
   - Format: MP4, 1080p, <50MB
   - Placement: Homepage hero section (autoplay, muted, loop)

2. **Program Showcase Videos** (Priority: HIGH)
   - One per program (Barber, CNA, HVAC, etc.)
   - Duration: 15-30 seconds each
   - Content: Students in training, equipment, instructors
   - Format: MP4, 720p, <25MB each
   - Placement: Program detail pages

3. **Student Testimonial Videos** (Priority: MEDIUM)
   - 3-5 testimonials
   - Duration: 30-60 seconds each
   - Content: Student interviews, before/after stories
   - Format: MP4, 1080p, <50MB each
   - Placement: Homepage, About page

4. **Facility Tour Video** (Priority: LOW)
   - Duration: 60-90 seconds
   - Content: Walkthrough of training facilities
   - Format: MP4, 1080p, <75MB
   - Placement: About page, Virtual tour section

### Video Hosting Options:

**Option 1: Self-Hosted** (Current setup)

- Location: `public/videos/`
- Pros: Full control, no external dependencies
- Cons: Larger file sizes, bandwidth costs

**Option 2: YouTube** (Recommended)

- Embed YouTube videos
- Pros: Free hosting, adaptive streaming, analytics
- Cons: YouTube branding, requires channel

**Option 3: Cloudflare Stream** (Professional)

- Pros: Fast delivery, adaptive streaming, no YouTube branding
- Cons: Paid service ($1/1000 minutes viewed)
