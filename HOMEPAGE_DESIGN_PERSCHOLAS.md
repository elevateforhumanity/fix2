## DESIGN PRINCIPLES OBSERVED

### 1. Whitespace & Breathing Room

**Generous Spacing:**
- Section padding: 80-100px vertical
- Element margins: 24-48px
- Card padding: 40px
- Grid gaps: 32-40px

**Why it works:**
- Content is easy to scan
- Hierarchy is clear
- Professional, premium feel
- Not overwhelming

### 2. Clear Visual Hierarchy

**Size Progression:**
- H1: 48px
- H2: 36px
- H3: 24px
- Body: 16-18px
- Small: 14px

**Weight Progression:**
- Headlines: Bold (700)
- Subheads: Semibold (600)
- Body: Regular (400)

**Color Hierarchy:**
- Primary: Black (#000)
- Secondary: Dark gray (#333)
- Tertiary: Medium gray (#666)
- Subtle: Light gray (#999)

### 3. Consistent Grid System

**Layout:**
- Max-width: 1200px
- Padding: 40-60px horizontal
- Columns: 12-column grid
- Gaps: 32px

**Breakpoints:**
- Mobile: < 768px (1 column)
- Tablet: 768-1024px (2 columns)
- Desktop: > 1024px (3-4 columns)

### 4. Color Palette

**Primary Colors:**
- White: #FFFFFF (backgrounds)
- Black: #000000 (text)
- Brand Blue: ~#0066CC (CTAs, accents)
- Light Gray: #F5F5F5 (section backgrounds)

**Usage:**
- High contrast (black on white)
- Minimal color usage
- Brand color for CTAs only
- Alternating section backgrounds

### 5. Typography System

**Font Family:**
- Sans-serif (likely custom or system font)
- Clean, modern, readable

**Scale:**
- 14px (small text)
- 16px (body)
- 18px (large body)
- 24px (H3)
- 36px (H2)
- 48px (H1)

**Line Heights:**
- Headlines: 1.2
- Body text: 1.6
- Tight for impact, loose for readability

### 6. Card Design

**Structure:**
```
┌─────────────────────┐
│  [Image/Icon]       │
│                     │
│  Headline           │
│  Body text...       │
│                     │
│  [CTA Link/Button]  │
└─────────────────────┘
```

**Styling:**
- White background
- 1px light gray border
- 8-12px border radius
- 40px padding
- Subtle shadow on hover
- Lift effect on hover

### 7. Button/CTA Design

**Primary Button:**
- Background: Brand blue
- Text: White
- Padding: 16px 32px
- Border radius: 8px
- Font: 16px semibold
- Hover: Darker shade

**Secondary Button:**
- Background: White

- Same padding/radius
- Hover: Light blue background

**Text Link:**
- Color: Brand blue
- Underline on hover
- Font: 16px semibold
- Arrow icon →

## COMPARISON: PER SCHOLAS VS YOUR SITE

### Per Scholas Strengths:

**1. Cleaner Hero**
- Video is the hero (not background)
- Simple headline above video
- No competing elements
- Clear focus

**2. Dual Audience Approach**
- Immediate path selection
- Clear "I want..." / "I need..." language
- Separate CTAs for each audience

**3. Social Proof Placement**
- Employer logos prominently displayed
- "Where our Alumni Work" section

- Hover reveals color

- Fewer sections on homepage
- Each section has one clear purpose
- No information overload
- Easy to scan

**5. Consistent Spacing**
- 80px section padding throughout
- 40px card padding
- 32-40px gaps
- Predictable rhythm

## KEY TAKEAWAYS

**What Makes Per Scholas Homepage Work:**

1. **Simplicity** - Each section has one clear purpose
2. **Whitespace** - 80px section padding creates breathing room
3. **Dual paths** - Clear separation for different audiences
4. **Social proof** - Employer logos prominently displayed
5. **Visual hierarchy** - Clear size/weight/color progression
6. **Consistent spacing** - Predictable rhythm throughout
7. **Minimal color** - Black/white with brand accent
8. **Large touch targets** - Easy to click/tap
9. **Mobile-first** - Responsive design that works everywhere
10. **Fast loading** - Optimized images and minimal scripts
