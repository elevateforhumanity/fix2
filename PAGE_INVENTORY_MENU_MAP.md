## INTERNAL LINKING RULES

### Every Page Must Have:
1. **Link to 2 sibling pages** (same level)
2. **Link to 1 parent page** (up one level)
3. **Link to 1 child page** (down one level)

### Example: Course Detail Page
- **Parent:** Course catalog
- **Siblings:** Other courses in same category
- **Children:** Course lessons, discussions
- **Related:** AI tutor, certificates, community

---

## NO ORPHAN PAGES CHECKLIST

### For Every Page, Verify:
- [ ] Reachable from header OR footer OR parent page

- [ ] Links back into the system

- [ ] Has proper meta tags for SEO

## DISCOVERY VERIFICATION

### Self-Check Questions:
1. **Is it reachable?** Header OR Footer OR Parent page
2. **Does it belong?** Programs, Learn, AI, or Platform
3. **Does it link back?** Internal linking present
4. **Is it indexed?** In sitemap (if public)
5. **Is it crawlable?** No orphan pages

### If YES to all → Page is discoverable ✅
