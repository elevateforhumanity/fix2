0Check If You Have Programs

Run this first to see if you already have programs in your database.

  1. Click the COPY SQL button below
  2. Go to Supabase Dashboard ‚Üí SQL Editor
  3. Paste the SQL (Ctrl+V or Cmd+V)
  4. Click RUN
  5. If you see programs, note their IDs. If not, continue to next step.
-- CHECK IF YOU HAVE PROGRAMS
SELECT 
  id,
  title,
  category,
  created_at
FROM programs
ORDER BY category, title;

-- If this returns NO ROWS, you need to create programs first!
-- If this returns rows, you already have programs and can use their IDs

1Get ETPL Program IDs RECOMMENDED

If you have ETPL (Eligible Training Provider List) programs, use this to get their IDs.

  1. Click COPY SQL
  2. Paste into Supabase SQL Editor
  3. Click RUN
  4. SAVE THE IDs from the results - you'll need them!
-- GET YOUR ETPL PROGRAM IDs
SELECT 
  id,
  title,
  slug,
  category,
  created_at
FROM programs
WHERE 
  -- ETPL programs typically have these characteristics
  funding @> ARRAY['WIOA']::text[]
  OR funding @> ARRAY['WRG']::text[]
  OR slug IN (
    'business-startup-marketing',
    'emergency-health-safety-tech',
    'barbering',
    'cosmetology',
    'esthetics',
    'cna',
    'medical-assistant',
    'phlebotomy',
    'hvac',
    'electrical',
    'welding',
    'it-support',
    'microsoft-office',
    'administrative-assistant',
    'culinary-arts',
    'retail-management',
    'janitorial-services'
  )
ORDER BY title;

2Get Partner Recommendations SMART

This shows which partner modules you should add to each program based on the program type.

  1. Click COPY SQL
  2. Paste into Supabase SQL Editor
  3. Click RUN
  4. See recommendations for each program
  5. Use the templates in Step 3 to add the recommended partners
-- WHAT PARTNER MODULES SHOULD I ADD?
SELECT 
  id,
  title,
  category,
  CASE 
    WHEN title ILIKE '%CNA%' OR title ILIKE '%nursing%' OR title ILIKE '%medical%' OR title ILIKE '%health%' 
      THEN '‚Üí Add: HSI (CPR), CareerSafe (Healthcare Safety), NDS (Drug-Free)'
    WHEN title ILIKE '%barber%' OR title ILIKE '%cosmetology%' OR title ILIKE '%beauty%' OR title ILIKE '%esthetic%'
      THEN '‚Üí Add: Milady RISE (Safety), NDS (Drug-Free)'
    WHEN title ILIKE '%hvac%' OR title ILIKE '%building%' OR title ILIKE '%maintenance%' OR title ILIKE '%electrical%'
      THEN '‚Üí Add: CareerSafe (OSHA 10), NDS (Drug-Free)'
    WHEN title ILIKE '%IT%' OR title ILIKE '%computer%' OR title ILIKE '%customer service%' OR title ILIKE '%business%'
      THEN '‚Üí Add: Certiport (Microsoft Office), NRF (Retail - optional)'
    WHEN title ILIKE '%janitor%' OR title ILIKE '%custodial%' OR title ILIKE '%cleaning%'
      THEN '‚Üí Add: JRI (Janitorial Training - FREE)'
    WHEN title ILIKE '%retail%' OR title ILIKE '%sales%'
      THEN '‚Üí Add: NRF RISE Up (Retail Training - FREE)'
    WHEN title ILIKE '%CDL%' OR title ILIKE '%driver%' OR title ILIKE '%transport%'
      THEN '‚Üí Add: NDS (DOT/CDL Drug & Alcohol)'
    ELSE '‚Üí Add: NDS (Drug-Free Workplace) - works for any program'
  END AS recommended_partners
FROM programs
ORDER BY category, title;

4Verify Partner Modules Added

Check that your partner modules were added successfully.

  1. Click COPY SQL
  2. Paste into Supabase SQL Editor
  3. Click RUN
  4. You should see your programs with their partner modules listed
-- Verify Partner Modules
SELECT 
  p.title AS program,
  p.category,
  epm.partner_name,
  epm.title AS module_title,
  epm.is_required,
  epm.hours
FROM programs p
LEFT JOIN external_partner_modules epm ON p.id = epm.course_id
ORDER BY p.category, p.title, epm.sort_order;
