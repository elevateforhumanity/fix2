# Elevate for Humanity - Site Improvement Plan

## Executive Summary
Transform elevateforhumanity.org into a professional, government-compliant, community-focused platform with consistent design, inspirational messaging, and polished user experience.

## Reference Sites Analyzed
- **IRS.gov** - Government compliance, clear navigation, professional typography
- **EmployIndy.org** - Community focus, inspirational messaging, clean structure
- **IvyTech.edu** - Educational institution standards, breadcrumbs, program structure
- **LearnWorlds.com** - LMS best practices, dashboard design
- **Milady.com** - Professional training standards, program presentation

## Critical Issues Identified

### 1. Typography
- **Current**: Inter (sans-serif), appears small
- **Target**: Times New Roman / Georgia (serif) for professional, government-compliant look
- **Action**: Replace font system-wide, increase base font size

### 2. Homepage Issues
- **Hero Image**: Barber picture improperly cropped
- **Black Sections**: Generic content, lacks inspiration
- **Flow**: Inconsistent spacing and rhythm
- **Action**: Fix image cropping, rewrite content with community focus

### 3. Navigation & Breadcrumbs
- **Current**: Breadcrumbs exist but may not be comprehensive
- **Target**: Clear hierarchy on every page
- **Action**: Audit and enhance breadcrumb implementation

### 4. Program Pages
- **Current**: Inconsistent structure, varying quality
- **Target**: Uniform template, professional images, clear outcomes
- **Action**: Create standard template, audit all program pages

### 5. Dashboards
- **Current**: Needs review
- **Target**: Clean, functional, user-friendly LMS experience
- **Action**: Audit student/admin dashboards, improve UX

### 6. Image Quality
- **Current**: Mixed quality, some improperly cropped
- **Target**: Professional, high-resolution, properly framed
- **Action**: Audit all images, fix cropping, replace low-quality

### 7. Content Consistency
- **Current**: Varying tone, depth, and structure
- **Target**: Consistent voice, inspirational, community-focused
- **Action**: Content audit, rewrite key sections

## Implementation Phases

### Phase 1: Foundation (Priority 1)
1. Change typography to Times New Roman/Georgia
2. Fix homepage hero image cropping
3. Rewrite black section content with inspiration
4. Increase base font sizes

### Phase 2: Structure (Priority 2)
1. Audit and fix breadcrumbs site-wide
2. Standardize program page template
3. Fix all program page images
4. Ensure consistent spacing/margins

### Phase 3: Content (Priority 3)
1. Rewrite homepage with storytelling
2. Audit all program descriptions
3. Add community impact messaging
4. Enhance "About" and mission content

### Phase 4: Dashboards (Priority 4)
1. Audit student dashboard
2. Audit admin dashboard
3. Improve LMS user experience
4. Add progress indicators

### Phase 5: Polish (Priority 5)
1. Image quality audit
2. Mobile responsiveness check
3. Accessibility compliance
4. Performance optimization

## Success Criteria
- Professional, government-compliant appearance
- Consistent design and flow throughout
- Inspirational, community-focused messaging
- Clear navigation and breadcrumbs
- High-quality images properly displayed
- Polished dashboards and user experience
- Mobile-friendly and accessible

- Phase 3: Medium-term (4-6 hours)
- Phase 4: Medium-term (3-5 hours)
- Phase 5: Ongoing (2-3 hours)
