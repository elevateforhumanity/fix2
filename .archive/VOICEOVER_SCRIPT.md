## Production Notes

### Background Music Recommendations:
1. **"Inspiring Corporate"** - Uplifting piano with strings
2. **"Hopeful Journey"** - Warm, motivational orchestral
3. **"New Beginnings"** - Soft, optimistic instrumental

### Audio Settings:
- Voice-over volume: 80%
- Background music volume: 20-30%
- Fade in/out: 1 second at start and end
- Format: MP3, 192kbps

### Recording Tips:
- Record in a quiet space with minimal echo
- Use a quality USB microphone (Blue Yeti, Audio-Technica AT2020)
- Speak clearly and warmly, as if talking to a friend
- Pause naturally between slides
- Smile while recording—it comes through in the voice

### File Placement:
Save the final audio file as:
```
/public/videos/voiceover.mp3
```

The slideshow component will auto-play this audio when the page loads (if browser allows).

## Alternative: Text-to-Speech Option

If you prefer AI voice generation, use:

Recommended voice settings:
- Voice: Rachel or Bella (ElevenLabs)
- Stability: 50%
- Clarity: 75%
- Style: Conversational, warm
