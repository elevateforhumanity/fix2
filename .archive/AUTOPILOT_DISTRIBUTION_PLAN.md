## üìù Module Creation Template

Each autopilot should create modules following this structure:

### Module Structure
```typescript
{
  id: "program-mod-X",
  title: "Module Title",
  description: "What students will learn",
  lessons: [
    {
      id: "program-X-Y",
      title: "Lesson Title",
      type: "reading" | "video" | "lab" | "quiz",
      durationMinutes: number,
      partnerRefCode?: "PARTNER-CODE" // if applicable
    }
  ]
}
```

### Lesson Types Distribution
- **Reading**: 40% (theory, concepts, procedures)
- **Video**: 20% (demonstrations, explanations)
- **Lab**: 30% (hands-on practice, skills)
- **Quiz**: 10% (knowledge checks, assessments)

### Module Content Requirements
1. **Introduction Module**: Overview, safety, tools, expectations
2. **Foundation Modules**: Core concepts and basic skills
3. **Advanced Modules**: Specialized techniques and applications
4. **Practice Modules**: Hands-on labs and skill development
5. **Assessment Module**: Exam prep, final skills check
