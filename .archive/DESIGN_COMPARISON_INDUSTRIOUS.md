# Design Comparison: Elevate vs. Enterprise Standards

## Hero Banner Sizing Standards

### Industry Best Practices:
- **Desktop:** 300-400px height
- **Mobile:** 250-300px height
- **Aspect Ratio:** 16:9 or 21:9 (ultrawide)
- **Image Quality:** 1920x600 minimum, q=90+

### Current Issues:
- ❌ Homepage hero: 400-700px (TOO TALL)
- ❌ Program pages: 400-600px (TOO TALL)
- ❌ Inconsistent sizing across pages

### Target Sizing:
- ✅ Desktop: 350px
- ✅ Tablet: 300px
- ✅ Mobile: 250px

---

## Page Structure Standards

### Every Page Should Have:

1. **Hero Banner** (350px max)
   - High-quality image
   - No text overlay
   - Clean, professional

2. **Hero Content Section** (below image)
   - Page title
   - Brief description
   - Primary CTA button
   - Secondary CTA (optional)

3. **Highlights Section** (with images)
   - 3-4 key features
   - Icon or image for each
   - Brief description

4. **Detailed Content**
   - Program details
   - What you'll learn
   - Career outcomes
   - Funding options

5. **Final CTA Section**
   - Strong call-to-action
   - Contact information
   - Apply button

---

## Image Quality Standards

### Hero Images:
- **Resolution:** 1920x600 (16:9 cropped)
- **Format:** WebP with JPEG fallback
- **Quality:** 90-95
- **File Size:** < 200KB
- **Content:** Bright, clear, professional

### Feature Images:
- **Resolution:** 800x600 (4:3)
- **Format:** WebP with JPEG fallback
- **Quality:** 85-90
- **File Size:** < 150KB

### Card Images:
- **Resolution:** 600x400 (3:2)
- **Format:** WebP with JPEG fallback
- **Quality:** 85
- **File Size:** < 100KB
