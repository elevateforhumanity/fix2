# How to Download and Split Images from Artlist

## Method 1: Browser Download (Easiest)

1. **Open the Artlist link in your browser** (while logged in)
2. **Right-click on the image**
3. **Select "Save Image As..."**
4. **Save to your Downloads folder**
5. **Drag the file into VSCode** → `/workspaces/fix2/public/media/`

## Method 2: Developer Tools (If right-click is disabled)

1. **Open the Artlist link**
2. **Press F12** (opens Developer Tools)
3. **Click "Network" tab**
4. **Refresh the page** (F5)
5. **Look for image files** (.png, .jpg, .webp)
6. **Right-click the image file** → "Open in new tab"
7. **Save the image** from the new tab

## Method 3: Screenshot (Last Resort)

1. **Open the Artlist link**
2. **Take a screenshot** (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
3. **Save the screenshot**
4. **Upload to VSCode**
