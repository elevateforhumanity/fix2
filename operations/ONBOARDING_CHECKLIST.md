# CUSTOMER ONBOARDING CHECKLIST

## 4-Week Process - Copy/Paste Ready

## PRE-ONBOARDING (Before Start Date)

### Sales Handoff

- [ ] MOU signed in DocuSign
- [ ] Payment method on file (Stripe)
- [ ] Customer info in CRM (HubSpot)
- [ ] Handoff email sent to onboarding team

### Account Setup

- [ ] Create organization account in system
- [ ] Generate admin login credentials
- [ ] Set up billing in Stripe
- [ ] Create Slack channel (optional)

### Schedule Calls

- [ ] Send welcome email with next steps
- [ ] Schedule onboarding kickoff call (60 min)
- [ ] Send calendar invite with Zoom link
- [ ] Send pre-call questionnaire

### Pre-Call Questionnaire (Send via Email)

```
Hi [Name],

Excited to get started! To make our kickoff call productive, please answer these questions:

1. How many programs do you offer?
2. How many students are currently enrolled?
3. Do you have existing student data to import? (Yes/No)
4. How many staff members need accounts?
5. What's your biggest priority? (Enrollment, Reporting, Mobile Access, Other)
6. Any specific questions or concerns?

Reply by [Date] so we can prepare.

Thanks!
Onboarding Team
```
