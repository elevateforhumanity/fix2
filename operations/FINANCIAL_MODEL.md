## UNIT ECONOMICS

### Customer Acquisition Cost (CAC)

```
Marketing & Sales Spend: $20K (Year 1)
New Customers: 10
CAC = $20K / 10 = $2,000 per customer
```

### Lifetime Value (LTV)

```
Average Revenue per Customer: $5K/month = $60K/year
Average Customer Lifetime: 3 years
Churn Rate: 10% per year

LTV = $60K × 3 years × (1 - 10% churn)
LTV = $162K per customer
```

### LTV:CAC Ratio

```
LTV:CAC = $162K / $2K = 81:1

Target: 3:1 minimum
Actual: 81:1 ✅ EXCELLENT
```

### Payback Period

```
CAC: $2,000
Monthly Revenue: $5,000

Payback = $2,000 / $5,000 = 0.4 months
Payback = 12 days ✅ EXCELLENT
```

### Gross Margin

```
Revenue: $5,000/month
COGS (hosting, support): $50/month

Gross Margin = ($5,000 - $50) / $5,000 = 99%
```

## BREAK-EVEN ANALYSIS

### Monthly Break-Even

```
Fixed Costs: $12K/month (Year 1)
Variable Costs: $50/customer/month
Revenue per Customer: $5,000/month

Break-Even = Fixed Costs / (Revenue - Variable Costs)
Break-Even = $12K / ($5K - $50)
Break-Even = 2.4 customers

RESULT: Need 3 customers to break even ✅
```
