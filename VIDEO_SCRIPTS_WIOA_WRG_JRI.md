## Production Notes

### Brand Guidelines:
- **Colors:** Use EFH brand palette (Red #E63946, Orange #F77F00, Teal #06A77D, Purple #7209B7, Blue #3A86FF)
- **Fonts:** Inter or similar clean sans-serif
- **Tone:** Professional, empowering, accessible
- **Music:** Uplifting, modern, not overpowering voiceover

### Accessibility:
- All videos MUST include captions (burned-in or SRT)
- Voiceover should be clear, moderate pace
- Visual text should be large enough to read on mobile
- High contrast for text overlays

### File Organization:
