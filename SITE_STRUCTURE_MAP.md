## ğŸ”— KEY USER JOURNEYS

### Journey 1: Prospective Student
1. Land on homepage (/)
2. Click [Explore Programs]
3. Browse programs (/programs)
4. Click program card â†’ /programs/medical-assistant
5. Read program details
6. Click [Check Funding Eligibility]
7. Fill out funding form (/funding)
8. Get connected to WorkOne/EmployIndy
9. Return to apply (/apply)
10. Get enrolled
11. Access LMS (/lms/dashboard)

### Journey 2: Current Student
1. Go to /lms
2. Click [Student Login]
3. Land on /lms/dashboard
4. Click "Continue Learning"
5. Enter course player (/lms/course/ma-101)
6. Complete lesson
7. Take quiz
8. Mark complete
9. Return to dashboard

### Journey 3: Employer
1. Land on homepage (/)
2. Click "For Employers"
3. Read employer benefits (/employers)
4. Click [Request Talent]
5. Fill out form
6. Get employer portal access
7. View candidates (/employer/candidates)
8. Request interviews

### Journey 4: Case Manager (WorkOne/EmployIndy)
1. Receive delegate portal access
2. Login to /delegate/dashboard
3. View caseload (/delegate/caseload)
4. Check student progress
5. Add case notes
6. Generate reports for agency
