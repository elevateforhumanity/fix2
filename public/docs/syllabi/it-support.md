# IT Support Specialist
## Program Syllabus

**Provider:** Elevate for Humanity  
**ETPL Provider ID:** 10000949  
**Program Code:** IT-SUPPORT  
**Duration:** 12-20 Weeks (320-480 Hours)  
**Delivery Method:** Online + Hands-On Labs  
**Funding:** Workforce Grants • WIOA • Employer Sponsors  

---

## Program Overview

Launch your tech career with CompTIA A+ certification! Learn computer hardware, software, networking, troubleshooting, and customer service. Program prepares for help desk, desktop support, and IT technician roles. No prior experience needed. Tech jobs start at $40K+ with room for growth.

---

## Learning Objectives

1. Install, configure, and troubleshoot computer hardware
2. Install and support operating systems (Windows, macOS, Linux)
3. Configure and troubleshoot network connections
4. Provide technical support and customer service
5. Apply security best practices
6. Use remote support tools and ticketing systems
7. Troubleshoot mobile devices and peripherals
8. Pass CompTIA A+ certification exams (220-1101 and 220-1102)

---

## Course Outline

### Module 1: Computer Hardware (60 hours)
- Computer components and architecture
- Motherboards, CPUs, and RAM
- Storage devices (HDD, SSD, NVMe)
- Power supplies and cooling systems
- Display technologies
- Peripheral devices
- Hardware installation and upgrades
- Troubleshooting hardware issues

### Module 2: Operating Systems (60 hours)
- Windows 10/11 installation and configuration
- macOS basics
- Linux fundamentals
- File systems and permissions
- User account management
- System utilities and tools
- OS troubleshooting
- Virtualization basics

### Module 3: Networking Fundamentals (60 hours)
- TCP/IP and OSI model
- IP addressing and subnetting
- DNS, DHCP, and network services
- Wireless networking (Wi-Fi standards)
- Network hardware (routers, switches, access points)
- Network troubleshooting
- Internet connectivity
- Network security basics

### Module 4: Mobile Devices (40 hours)
- iOS and Android operating systems
- Mobile device configuration
- Email and synchronization
- Mobile security
- App installation and management
- Troubleshooting mobile issues
- Mobile device management (MDM)

### Module 5: Security (50 hours)
- Security threats and vulnerabilities
- Malware prevention and removal
- Authentication and access control
- Encryption and data protection
- Physical security
- Wireless security
- Security best practices
- Incident response basics

### Module 6: Software Troubleshooting (40 hours)
- Application installation and removal
- Windows troubleshooting tools
- Boot issues and recovery
- Performance optimization
- Driver updates
- Software compatibility
- Backup and recovery

### Module 7: Cloud Computing & Virtualization (30 hours)
- Cloud service models (IaaS, PaaS, SaaS)
- Cloud storage solutions
- Virtual machines and hypervisors
- Remote desktop services
- Cloud security considerations

### Module 8: Customer Service & Professionalism (30 hours)
- Communication skills
- Ticketing systems
- Remote support tools
- Documentation best practices
- Time management
- Professional conduct
- Handling difficult customers

### Module 9: Hands-On Labs (80-160 hours)
- Hardware assembly and disassembly
- OS installation and configuration
- Network setup and troubleshooting
- Security implementation
- Real-world troubleshooting scenarios
- Help desk simulations

## Certifications Earned

1. **CompTIA A+ Core 1 (220-1101)** - Hardware and networking

## Certification Pathway

- **CompTIA Network+** - Advanced networking
- **CompTIA Security+** - Cybersecurity fundamentals
- **Microsoft certifications** - Windows Server, Azure
- **Cisco CCNA** - Networking specialist
