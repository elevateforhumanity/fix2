**International Certification:**

- Foreign earned income

- Dual-status returns
- Treaty benefits

### Certification Levels

**Basic:**

- Required for all volunteers
- Covers most common returns
- 80% passing score required
- Valid for current tax year

**Advanced:**

- For complex returns
- Requires Basic certification
- Additional training modules
- Higher skill level

**Military:**

- Military-specific issues
- Requires Basic certification
- Special scenarios
- Military base volunteers

**International:**

- Foreign income and credits
- Requires Advanced certification
- Complex situations
- Expat returns
