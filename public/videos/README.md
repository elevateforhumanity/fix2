## Video Specifications

### Homepage Hero Video
- **Duration**: 30-60 seconds
- **Resolution**: 1920x1080 (1080p)
- **Format**: MP4 (H.264)
- **Bitrate**: 5-8 Mbps
- **File Size**: <50MB
- **Audio**: Optional (will autoplay muted)
- **Content**: Overview of programs, students in training

### Program Showcase Videos
- **Duration**: 15-30 seconds
- **Resolution**: 1280x720 (720p) or 1920x1080 (1080p)
- **Format**: MP4 (H.264)
- **Bitrate**: 3-5 Mbps
- **File Size**: <25MB each
- **Audio**: Optional
- **Content**: Students in training, equipment, instructors

### Testimonial Videos
- **Duration**: 30-60 seconds
- **Resolution**: 1280x720 (720p)
- **Format**: MP4 (H.264)
- **Bitrate**: 3-5 Mbps
- **File Size**: <30MB each
- **Audio**: Required (student speaking)
- **Content**: Student interviews, success stories

## Upload Instructions

1. Place video files in appropriate directories
2. Ensure filenames match exactly (lowercase, no spaces)
3. Test video playback in browser
4. Verify file sizes are within limits
5. Add captions/subtitles if needed (VTT format)
