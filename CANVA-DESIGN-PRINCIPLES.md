# Canva Design Principles to Apply

## What Makes Canva's Design Excellent (and FREE)

### 1. CLEAN, MINIMAL INTERFACE

- Lots of white space
- No clutter
- Clear visual hierarchy
- One primary action per section

### 2. VIBRANT, FRIENDLY COLORS

- Purple primary (#7D2AE8)
- Bright accent colors
- Gradients used sparingly
- High contrast for accessibility

### 3. SMOOTH ANIMATIONS

- Subtle hover effects
- Smooth transitions (200-300ms)
- Delightful micro-interactions
- No jarring movements

### 4. PERFECT TYPOGRAPHY

- Clear hierarchy (3-4 sizes max)
- Generous line spacing
- Easy to scan
- Bold headlines, regular body

### 5. CARD-BASED LAYOUT

- Everything in cards
- Consistent card styling
- Hover effects on cards
- Clear card boundaries

### 6. EXCELLENT MOBILE EXPERIENCE

- Mobile-first design
- Touch-friendly buttons (44px min)
- Simplified mobile navigation
- Fast loading on mobile

### 7. SMART USE OF IMAGERY

- High-quality photos
- Consistent image treatment
- Images support content
- Lazy loading

### 8. CLEAR CALLS TO ACTION

- One primary CTA per section
- High contrast buttons
- Action-oriented copy
- Obvious next steps

### 9. PROGRESSIVE DISCLOSURE

- Show what's needed, hide complexity
- Expandable sections
- Tooltips for help
- Guided workflows

### 10. DELIGHTFUL DETAILS

- Loading animations
- Success celebrations
- Error messages are friendly
- Empty states are helpful

## Apply to Elevate Site

### Homepage

- Use card-based layout for programs
- Add subtle hover animations
- Simplify to one CTA per section
- Add more white space
- Use vibrant accent colors for badges

### Programs Page

- Card grid like Canva templates
- Hover preview effect
- Quick filters at top
- Smooth transitions

### Application Flow

- Progress indicator like Canva
- One question per screen option
- Celebrate completion
- Friendly error messages

### Student Portal

- Dashboard cards like Canva
- Quick actions prominent
- Recent activity feed
- Smooth page transitions

### Overall

- Reduce visual noise
- Increase white space
- Add delightful animations
- Make CTAs more prominent
- Improve mobile experience
