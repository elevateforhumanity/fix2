## üè¢ Testing Center Setup

### Physical Requirements

**Testing Room:**
- Quiet, secure space
- Individual workstations
- Adequate lighting
- Climate controlled
- No distractions

**Computers:**
- Windows 10/11 or macOS
- Minimum 4GB RAM (8GB recommended)
- Stable internet connection
- Webcam (for remote proctoring if applicable)
- Headphones (for some exams)

**Proctor Station:**
- Separate computer for proctor
- View of all test-takers
- Access to Certiport portal
- Phone for support

**Security:**
- No phones/devices allowed
- No reference materials
- Monitored environment
- Secure check-in/check-out

### Software Requirements

**Certiport Testing Software:**
- Download from Certiport portal
- Install on all testing computers
- Regular updates required
- Test connection before exams

**System Check:**
- Run Certiport system check tool
- Verify internet speed
- Test exam delivery
- Confirm proctor access

## üìä Exam Administration

### Scheduling Exams

**Options:**
1. **Walk-in testing** - Students schedule directly
2. **Group testing** - Schedule class sessions
3. **Individual appointments** - One-on-one testing

**Best practice:** Schedule group sessions for classes

### Exam Process

1. **Before Exam:**
   - Student registers in Certiport portal
   - Pays exam fee (or voucher)
   - Schedules exam time
   - Receives confirmation

2. **Day of Exam:**
   - Student arrives 15 min early
   - Check ID and confirmation
   - Review exam rules
   - Log into testing computer
   - Launch exam

3. **During Exam:**
   - Proctor monitors
   - No assistance allowed
   - Handle technical issues only
   - Time limit enforced

4. **After Exam:**
   - Immediate score report
   - Pass/Fail notification
   - Certificate issued (if passed)
   - Results sent to Certiport

### Retake Policy:
- Failed exams can be retaken
- Waiting period may apply
- Additional fee required
- Study resources provided
