## üìê IMAGE SPECIFICATIONS

### Technical Requirements

**File Formats:**
- Photos: JPG (optimized, 80-90% quality)
- Graphics: PNG (transparent backgrounds)
- Logos: SVG (vector) + PNG fallback
- Icons: SVG preferred

**Size Guidelines:**
- Hero images: 1920x1080px (16:9)
- Program cards: 800x600px (4:3)
- Thumbnails: 400x300px (4:3)
- Portraits: 600x600px (1:1)
- Banners: 1920x600px (wide)

**Optimization:**
- Max file size: 500KB for photos
- Max file size: 100KB for graphics
- Use WebP format where supported
- Provide 2x retina versions

**Accessibility:**
- All images need descriptive alt text
- Minimum color contrast: 4.5:1
- No text embedded in images (use overlays)

- "hands-on learning"
