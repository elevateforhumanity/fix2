## üîë HOW TO ACCESS REPORTS

### For Admin/Staff (Monster Pull Reports):

1. **Login**: Go to https://www.elevateforhumanity.org/login
2. **Navigate**: After login, go to https://www.elevateforhumanity.org/admin/reports
3. **Filter**:
   - Select program (or ALL)
   - Set date range (optional)
   - Click "Refresh" or "Load"
4. **Export**:
   - Click "Download" or "Export to CSV" button
   - Open in Excel/Google Sheets

### For Program Holders:

1. **Login**: Go to https://www.elevateforhumanity.org/login
2. **Dashboard**: Automatically redirected to program holder dashboard
3. **View Students**: See your assigned students and their progress
4. **Reports**: Access reports specific to your students

### For Delegates/Case Managers:

1. **Login**: Go to https://www.elevateforhumanity.org/login
2. **Dashboard**: Automatically redirected to delegate dashboard
3. **Reports**: Go to https://www.elevateforhumanity.org/delegate/reports
4. **Export**: Use export functionality to download data

## üîí AUTHENTICATION & SECURITY

### Password Reset
**URL**: https://www.elevateforhumanity.org/forgotpassword

If you forgot your password, use this page to reset it.

### Unauthorized Access
If you try to access a page without proper permissions, you'll be redirected to:
**URL**: https://www.elevateforhumanity.org/unauthorized

### Role-Based Access
- Each user role has specific permissions
- You can only access pages/data relevant to your role
- Admin has access to all features
- Program holders see only their assigned students
- Students see only their own data

## üéØ COMMON TASKS

### Pull a Report for All Students (Admin):
1. Login at `/login`
2. Go to `/admin/reports`
3. Select "ALL" for program
4. Set date range (optional)
5. Click "Download" or "Export to CSV"

### Check Student Progress (Program Holder):
1. Login at `/login`
2. View dashboard at `/program-holder/dashboard`
3. Click on student name to see details
4. View grades at `/program-holder/grades`

### View My Courses (Student):
1. Login at `/login`
2. View dashboard at `/student/dashboard`
3. Click "My Courses" or go to `/student/courses`
4. Click on course to access materials
