## ğŸ”§ HOW TO ADD TO DURABLE

### For Navigation Menu:

1. Go to Durable website editor
2. Click on Navigation/Menu
3. Add "Link" element
4. Paste the URL from above
5. Add link text (e.g., "Programs", "About", etc.)

### For Buttons:

1. Add a Button element
2. Click "Link" or "URL"
3. Paste the URL (e.g., `/apply`)
4. Style the button

### For Dropdown Menus:

1. Create parent menu item (e.g., "Programs")
2. Add child items underneath
3. Link each child to specific program page

## ğŸ’¡ PRO TIPS

1. **Use relative URLs** in Durable if possible:
   - `/programs` instead of full URL
   - Durable will auto-add your domain

2. **Set link targets:**
   - Internal links: Same tab
   - External links: New tab (`target="_blank"`)

3. **Add tracking:**
   - Use UTM parameters for marketing links
   - Example: `/apply?utm_source=durable&utm_medium=nav`

4. **Mobile optimization:**
   - Test all links on mobile
   - Ensure dropdown menus work on touch

5. **Update regularly:**
   - Add new programs as they launch
   - Update seasonal CTAs
   - Keep social links current
