## üé® IMAGE SPECIFICATIONS

### General Requirements
- **Format:** JPG (optimized)
- **Max file size:** 500KB per image
- **Color space:** sRGB
- **Quality:** 80-90%

### Size Guidelines
- **Hero images:** 1920x1080px or 1920x600px
- **Card images:** 800x600px or 600x400px
- **Thumbnails:** 400x300px
- **Portraits:** 600x600px (square)
- **Video placeholders:** 1280x720px (16:9)

### Content Guidelines
- **Real people:** Use real students/staff when possible
- **Diverse:** Show diversity in age, race, gender
- **Professional:** Clean, well-lit, high quality
- **Bright:** Avoid dark, moody images
- **Action:** Show people doing things, not just posing
- **Authentic:** Avoid overly staged stock photos


**If you use the script:**
1. Place images with exact names above
2. Run script in Gitpod
3. Placeholders automatically replaced

**If you don't use the script:**
1. Place images with exact names above
2. Update image paths in code manually
3. Use `src="/images/filename.jpg"` directly

## üí° SOURCING STRATEGIES

### Option 1: Professional Photography ($3,000-5,000)
- Hire photographer
- Shoot real students/facilities
- Get model releases
- **Best for:** Authenticity and brand

### Option 2: Stock Photography ($500-1,000)
- Use Unsplash, Pexels, Adobe Stock
- Search: "adult education", "vocational training", "workforce"
- **Best for:** Quick start, budget-friendly

### Option 3: AI-Generated ($100-300)
- Use Midjourney, DALL-E, Stable Diffusion
- Generate specific scenarios
- **Best for:** Concepts, placeholders

### Option 4: Hybrid (Recommended) ($1,500-2,500)
- 15 custom photos (real students)
- 15 stock photos (high quality)
- 10 AI-generated (concepts)
- **Best for:** Balance of quality and cost
