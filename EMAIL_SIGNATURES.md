## Installation Instructions

### Gmail
1. Open Gmail Settings (gear icon > See all settings)
2. Scroll to "Signature" section
3. Click "Create new" and name it
4. Paste HTML template
5. Replace placeholders with your information
6. Set as default signature
7. Save changes

### Outlook (Desktop)
1. File > Options > Mail > Signatures
2. Click "New" to create signature
3. Paste HTML in editor
4. Replace placeholders with your information
5. Set as default for new messages
6. Click OK

### Outlook (Web)
1. Settings (gear icon) > View all Outlook settings
2. Mail > Compose and reply
3. Under "Email signature", paste HTML
4. Replace placeholders with your information
5. Check "Include signature on new messages"
6. Save

### Apple Mail
1. Mail > Preferences > Signatures
2. Click "+" to add new signature
3. Paste HTML template
4. Replace placeholders with your information
5. Drag signature to desired email account

## Customization Guide

### Replace These Placeholders
- `[Your Name]` - Your full name
- `[Your Title]` - Your job title
- `[your.email]` - Your email username (before @)
- `[Your Phone]` - Your phone number (optional)

### Logo Image
Current logo URL: `https://www.elevateforhumanity.org/logo.png`

If you need a different size:
- Small: 60x60px
- Standard: 80x80px
- Large: 100x100px

### Color Scheme
- Primary Green: `#10b981`
- Text Dark: `#333`
- Text Medium: `#666`
- Text Light: `#999`

### Social Media Icons
Add your social profiles:
- LinkedIn: Update company URL
- Twitter: Update handle
- Facebook: Add if needed

## Best Practices

1. **Keep it Simple**: Don't overload with information
2. **Mobile-Friendly**: Test on mobile devices
3. **Professional**: Use consistent branding
4. **Updated**: Keep contact info current
5. **Legal**: Include required disclaimers if needed

## Testing Checklist

Before using your signature:
- [ ] All links work correctly
- [ ] Logo image displays
- [ ] Text is readable on mobile
- [ ] No placeholder text remains
- [ ] Email address is correct
- [ ] Website URL is correct
- [ ] Displays correctly in Gmail
- [ ] Displays correctly in Outlook
- [ ] Displays correctly on mobile
