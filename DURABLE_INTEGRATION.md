# Embed LMS on Durable

```html
<div id="lms"></div>
<script src="https://app.elevateforhumanity.org/embed/lms-widget.js"></script>
<script>
  ElevateLMS.embed('lms');
</script>
```

OR

```html
<button onclick="ElevateLMS.openCourses()">Browse Courses</button>
<script src="https://app.elevateforhumanity.org/embed/lms-widget.js"></script>
```
