## 1. Getting Started

Welcome to Elevate for Humanity! This guide will help you get started as an instructor.

## 2. Your Dashboard

After logging in, you'll see:
- **My Courses**: Courses you're teaching
- **Students**: Enrolled students
- **Assignments**: Pending grading
- **Schedule**: Upcoming classes

## 3. Managing Courses

### 3.1 Course Structure

Each course contains:
- **Modules**: Major topics
- **Lessons**: Individual learning units
- **Assessments**: Quizzes and exams
- **Resources**: Files and links

### 3.2 Adding Content

1. Navigate to your course
2. Click "Add Module"
3. Add lessons within modules
4. Upload resources (videos, PDFs, SCORM packages)

### 3.3 Creating Assessments

1. Go to "Assessments"
2. Click "Create Assessment"
3. Choose type:
   - Quiz (auto-graded)
   - Assignment (manual grading)
   - Exam (proctored)
4. Add questions or instructions
5. Set due date and passing score

## 4. Student Management

### 4.1 Viewing Student Progress

- Access "Students" tab
- View completion percentages
- Check assessment scores
- Review time spent

### 4.2 Grading Assignments

1. Go to "Assignments"
2. Select pending submissions
3. Review student work
4. Provide feedback
5. Assign grade

### 4.3 Communication

- Send announcements to all students
- Message individual students
- Schedule office hours
- Post in discussion forums

## 5. Live Sessions

### 5.1 Scheduling Live Classes

1. Navigate to "Live Sessions"
2. Click "Schedule Session"
3. Choose date/time
4. Select Zoom integration
5. Students receive notification

### 5.2 Recording Sessions

- Enable recording in Zoom
- Recordings auto-upload to course
- Students can review later

## 6. Attendance Tracking

1. Access "Attendance"
2. Mark students present/absent
3. Export attendance reports
4. Required for WIOA compliance

## 7. Reports and Analytics

### 7.1 Course Analytics

- Enrollment trends
- Completion rates
- Average scores
- Time to completion

### 7.2 Student Analytics

- Individual progress
- Engagement metrics
- At-risk students
- Success predictions

## 8. Best Practices

1. **Regular Updates**: Post announcements weekly
2. **Timely Grading**: Grade within 48 hours
3. **Engagement**: Respond to questions promptly
4. **Accessibility**: Use captions, alt text
5. **Feedback**: Provide constructive comments
