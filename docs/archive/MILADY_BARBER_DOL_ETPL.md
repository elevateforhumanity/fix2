#### Program Structure

**Total Hours:** 2,000 hours

- **On-the-Job Learning (OJL):** 1,600 hours (80%)
- **Related Technical Instruction (RTI):** 400 hours (20%)

#### Competency-Based Progression

**Apprentice Levels:**

1. **Level 1 (0-500 hours)**
   - Wage: Starting at $12/hour
   - Focus: Basic techniques, sanitation, client interaction
2. **Level 2 (500-1,000 hours)**
   - Wage: $14/hour
   - Focus: Independent services, advanced techniques
3. **Level 3 (1,000-1,500 hours)**
   - Wage: $16/hour
   - Focus: Complex services, chemical applications
4. **Level 4 (1,500-2,000 hours)**
   - Wage: $18/hour
   - Focus: Master-level work, state board preparation

## Certification Pathway

### Program Completion Requirements

1. **Complete 2,000 Hours**
   - 1,600 hours on-the-job training
   - 400 hours classroom instruction
   - Documented attendance and performance

2. **Pass Module Assessments**
   - Written exams for each module (75% minimum)
   - Practical skill demonstrations
   - Portfolio of work

3. **Mid-Program Evaluation**
   - Comprehensive assessment at 1,000 hours
   - Skills verification
   - Progress review

4. **Final Practical Examination**
   - Complete service demonstration
   - Timed performance
   - Quality assessment (85% minimum)

### Credentials Earned

**Upon Program Completion:**

1. **Milady Barber Apprenticeship Certificate**
   - Issued by: Elevate for Humanity
   - Verifies: Completion of 2,000-hour program
   - Includes: Milady curriculum completion

2. **DOL Registered Apprenticeship Certificate**
   - Issued by: U.S. Department of Labor
   - Verifies: Completion of registered apprenticeship
   - Portable: Recognized nationwide

3. **Eligibility for Indiana State Barber License**
   - Qualifies to sit for state board exam
   - Written and practical examination
   - Upon passing: Licensed Barber in Indiana
