# Upload Your 4 Homepage Images Here

Place these 4 images in this folder with exact filenames:

1. **home-hero-top-gradient.jpg** - TOP homepage hero banner (gradient)
2. **home-hero-second-programs.jpg** - 2nd hero banner (program grid)
3. **founder-bio-desk-side.jpg** - Side photo for founder bio page (at desk)
4. **founder-home-fullbody.jpg** - Full-body founder image for homepage

**Note:** Certificate of completion is separate - only used when students complete courses.

## After uploading, run:

```bash
bash scripts/setup-efh-images.sh
```

This will copy them to `public/images/efh/` with the correct names.
