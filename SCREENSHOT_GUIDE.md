## ğŸ› ï¸ Screenshot Tools & Methods

### Method 1: Browser DevTools (Recommended)
1. Open the page in Chrome/Edge
2. Press `F12` to open DevTools
3. Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
4. Type "screenshot" and select:
   - "Capture full size screenshot" - for full page
   - "Capture screenshot" - for visible area
   - "Capture node screenshot" - for specific element

### Method 2: Browser Extensions
- **Awesome Screenshot** (Chrome/Firefox)
- **Fireshot** (Chrome/Firefox)
- **Nimbus Screenshot** (Chrome)

### Method 3: Automated with Puppeteer
We have a script available:
```bash
node playwright_scripts/capture-screenshots.js
```

### Method 4: Manual Screenshots
- **Windows**: `Win + Shift + S`
- **Mac**: `Cmd + Shift + 4`
- **Linux**: `PrtScn` or Screenshot tool

## ğŸ¨ Quality Guidelines

1. **Clean Data**: Use realistic sample data, no Lorem Ipsum
2. **Consistency**: Same user/data across related screenshots
3. **Lighting**: Ensure good contrast and readability
4. **Focus**: Highlight key features and functionality
5. **Context**: Show enough UI to understand the page purpose
6. **No Sensitive Data**: Remove any real user information
