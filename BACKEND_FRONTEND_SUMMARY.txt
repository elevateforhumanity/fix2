💡 HOW TO USE THIS TEMPLATE

For Node.js Projects:
  1. Copy templates/gitpod-nodejs.yml to .gitpod.yml
  2. Copy .vscode/ and .devcontainer/ to your project
  3. Add your application code (src/, components/, etc.)
  4. Customize configuration for your needs

For Python Projects:
  1. Copy templates/gitpod-python.yml to .gitpod.yml
  2. Copy .vscode/ and .devcontainer/ to your project
  3. Add your application code (app/, api/, etc.)
  4. Customize configuration for your needs

For Full-Stack Projects:
  1. Copy templates/gitpod-fullstack.yml to .gitpod.yml
  2. Copy .vscode/ and .devcontainer/ to your project
  3. Create frontend/ and backend/ directories
  4. Add your application code
  5. Customize configuration for your needs
