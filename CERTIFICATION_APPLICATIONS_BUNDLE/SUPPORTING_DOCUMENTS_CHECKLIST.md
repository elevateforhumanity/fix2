## DOCUMENT ORGANIZATION TIPS

**Create Folders:**

```
Certifications/
├── Business_Formation/
├── Tax_Returns/
├── Financial_Statements/
├── Personal_Documents/
├── Licenses_Registrations/
├── Existing_Certifications/
├── Contracts_Agreements/
├── Employee_Information/
├── Photos/
└── Supporting_Letters/
```

**Digital Copies:**

- Scan all documents to PDF
- Name files clearly (e.g., "2024_Form990_SelfishInc.pdf")
- Keep organized in cloud storage
- Have backups

**Physical Copies:**

- Keep originals in secure location
- Create binder with copies for reference
- Organize by certification type
- Update regularly
