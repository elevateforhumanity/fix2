1. **Applicant Applies**
   - Fills out 6-step application
   - Uploads all required documents

   - Reference checks

5. **Onboarding**
   - Welcome email
   - Training assignments
   - Drake software access
   - Office setup

6. **Training**
   - Enroll in required courses
   - Complete Drake Fundamentals
   - Pass certification exams
   - Get certificates

7. **Start Work**
   - Assigned to tax returns
